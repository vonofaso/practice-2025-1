Журнал прогресса

Старт проекта и планирование

Прошло первое собрание команды, на котором мы:

  • Познакомились с членами команды и распределили роли
  • Изучили материалы от предыдущей команды разработчиков
  • Составили предварительный план работы на семестр
  • Определили ключевые задачи для каждого участника

Команда разработчиков начала изучение SwiftUI и API университета, а дизайнеры приступили к анализу существующего интерфейса личного кабинета.

Первые прототипы и изучение API

За прошедшие недели мы добились следующих результатов:

  • Дизайнеры подготовили первые прототипы основных экранов приложения
  • Разработчики изучили структуру API и успешно получили тестовые данные
  • Создана базовая архитектура приложения в Xcode
  • Реализован экран авторизации с простой валидацией

Основные сложности возникли с документацией API, часть endpoints оказалась устаревшей. Решено запросить актуальную информацию у администрации университета.

Реализация основных экранов

Текущий прогресс разработки:

  • Завершена работа над экраном расписания (80% функционала)
  • Реализована загрузка данных об успеваемости из API
  • Добавлена темная тема с сохранением настроек
  • Улучшена производительность при работе с сетью

Команда тестирования начала проверку реализованного функционала. Обнаружены проблемы с отображением данных при плохом интернет-соединении - работаем над решением.

Дизайнеры завершили работу над всеми основными экранами и перешли к созданию анимаций переходов между ними.
